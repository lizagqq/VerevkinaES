### Выводы

1. Ускорение 11.29x на 16 процессах
2. Эффективность 70.6% (сравнимо с явной схемой: 69.6%)
3. Неявная схема требует в 60 раз меньше временных шагов

## Теория

### Метод ROS1

```
[E - α τ f_y] w₁ = f(y_m, t_m + τ/2)
y_(m+1) = y_m + τ · Re(w₁)
```

### Преимущества неявной схемы

- Безусловно устойчива (нет ограничения на τ)
- Больший шаг по времени (τ_implicit ≈ 20 × τ_explicit)
- Меньше временных слоёв для достижения T

### Сравнение с явной схемой (Lab8)

| Характеристика | Явная (Lab8) | Неявная (Lab9) |
|----------------|--------------|----------------|
| Шаг по времени | ~2×10^(-5) | ~4×10^(-4) |
| Временных слоёв | 300000 | 5000 |
| Ускорение (16P) | 11.14x | 11.29x |
| Эффективность | 69.6% | 70.6% |
